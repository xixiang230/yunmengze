# 添加RSS
* 使用npm安装插件  
```
npm install hexo-generator-feed
```
* 在配置文件中引用插件  
```
在hexo/_config.yml添加  
plugins:  
-hexo-generator-feed  
```

# 添加Sitemap  
* 使用npm安装插件  
```
npm install hexo-generator-sitmap  
```
* 在配置文件中引用插件  
```  
在hexo/_config.yml中添加  
plugins:  
- hexo-generator-sitemap  
```
